# A股选股网站 - StockSelector

## 项目概述
专业的A股选股分析平台，提供实时数据分析、智能选股策略和投资组合管理功能。

## 技术栈
- **前端**: React + TypeScript + Ant Design + ECharts
- **后端**: FastAPI + Python + PostgreSQL + Redis
- **部署**: Docker + Nginx + 云服务器

## 核心功能
1. 🔍 智能选股系统
2. 📊 数据可视化分析
3. 💼 投资组合管理
4. ⚡ 实时行情推送
5. 📈 策略回测功能

## 项目结构
```
stock-selector/
├── frontend/          # React前端应用
├── backend/           # FastAPI后端服务
├── data/             # 数据处理模块
├── deploy/           # 部署配置
└── docs/             # 项目文档
```
